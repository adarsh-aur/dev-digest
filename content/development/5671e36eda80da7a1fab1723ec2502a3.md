## Summary

Unity and Unreal engines have different garbage collection (GC) systems,Unity uses C# and .NET GC, while Unreal uses C++ and a custom mark-and-sweep GC,Unreal's GC only tracks UObject subclasses and references marked with UPROPERTY(),In Unity, the GC is a performance concern, while in Unreal it's a correctness concern,Understanding the differences in GC systems is crucial for avoiding bugs and performance issues
