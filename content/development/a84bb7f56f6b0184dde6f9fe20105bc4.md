## Summary

Apache Iceberg exposes internal metadata as queryable virtual tables,The $snapshots table lists every snapshot in the table's history,The $history table shows the timeline of which snapshot was current at each point in time,The $files table lists every data file in the current snapshot with detailed statistics,The $manifests table lists the manifest files for the current snapshot,The $partitions table provides statistics per partition,These tables can be used for debugging, auditing, and monitoring
